# 合规逻辑模型详解

## 法规依据

- 《中华人民共和国药品管理法》（2019修订）
- 《医药代表备案管理办法（试行）》（国家药监局 2020）
- 《关于进一步规范医药代表从业行为的通知》
- 《医疗机构工作人员廉洁从业九项准则》

## 一、身份偏移逻辑

### 合规定位

医药代表的法定职责：
1. 在医疗机构开展学术推广活动
2. 向医务人员传递药品相关的学术信息
3. 收集、反馈药品临床使用情况及不良反应信息

### 违规表现

| 违规行为 | 对应条款 | 严重性 |
|----------|----------|--------|
| 承担药品销售任务 | 备案管理办法第13条 | 高 |
| 收款和处理购销票据 | 备案管理办法第13条 | 高 |
| 进行商业贿赂 | 药品管理法第88条 | 高 |
| 参与统方 | 备案管理办法第13条 | 高 |
| 岗位名称含"销售"但实际从事处方药推广 | 身份偏移 | 中 |

### 判定标准

- 岗位名称含"销售代表"、"销售经理"且涉及处方药 → 不合规
- 职责含"完成销售指标"、"销售任务" → 不合规
- 职责含"学术推广"、"信息传递" → 合规

## 二、处方干预逻辑

### 核心禁令

严禁以任何形式影响医生的独立处方决策。

### 违规表现

| 违规行为 | 说明 | 严重性 |
|----------|------|--------|
| 统方 | 收集特定医生处方数据 | 高 |
| 处方量追踪 | 追踪个人开方量 | 高 |
| 推广与处方量挂钩 | 按处方量给予回报 | 高 |
| 设定处方量指标 | 考核含处方量 | 高 |
| "上量"、"带量" | 暗示增加处方量 | 中 |
| "提高产出" | 可能暗示增加处方 | 中 |

## 三、利益交换逻辑

### 核心禁令

严禁以任何形式的利益换取药品销量或进院。

### 违规表现

| 违规行为 | 说明 | 严重性 |
|----------|------|--------|
| 回扣/返点 | 直接商业贿赂 | 高 |
| 捐赠换销量 | 预设商业条件的捐赠 | 高 |
| 过票/虚开发票 | 违法行为 | 高 |
| 客情维护费 | 变相利益输送 | 中 |
| 业务招待礼品 | 超出合规范围的馈赠 | 中 |

## 四、非学术化推广逻辑

### 核心要求

所有推广活动必须具有学术性质，在合规场景进行。

### 合规场景

- 医疗机构内的学术讲座
- 正规学术会议
- 线上学术平台

### 违规场景

- 餐厅（餐桌会）
- 娱乐场所
- 旅游景点
- 任何非学术目的的聚会

## 五、科学严谨性逻辑

### 核心要求

所有学术内容必须基于循证医学证据，不得夸大或误导。

### 违规表现

| 违规行为 | 说明 | 严重性 |
|----------|------|--------|
| 使用"根治"、"首选" | 绝对化疗效宣称 | 中 |
| 隐瞒不良反应 | 信息不完整 | 高 |
| "安全无毒" | 误导性宣称 | 高 |
| 未经验证的疗效宣称 | 缺乏循证依据 | 中 |
